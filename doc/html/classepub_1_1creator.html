Detailed Description

An individual or organizational creator.

The creator is the primary source for the content: the person responsible for its creation. It is usually a name in display form. There are two refinements that can be added to a creator. The file-as refinement is a string that is appropriate for sorting, such as "last-name, first-name." The role refinement is the MARC relator code describing how the creator was involved: "aut" for author, "ill" for illustrator, and so on.
